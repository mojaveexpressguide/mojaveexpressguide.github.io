MEG - Content

As you did with Viva New Vegas create new separators for the above categories.

Distributed Necklaces and Chains​

Adds Seam Concealing Chains and Pendants to all NPC's throughout the wasteland.

Installation:

  • Main Files - Distributed Necklaces and Chains

Spice of Life - Vanilla Bodies Patched​

A custom fixed vanilla body version of Spice of Life which adds a variety of clothing and armour to the game that will be distributed to various NPC's.

Installation:

  • Main Files - Spice of Life Vanilla Bodies Patched.7z
Installation:

Download and install manually in MO2

  • Main Files - KARPSOL - Kabyi Armor Replacement Plugin Spice Of Life

CIBS - Customizable Integrated Backpack System​

This mod adds integrated backpacks into New Vegas and distributes them to NPC's, Vendors and throughout the world. You will also need to download Canvas Backpacks - FNV - TTW below.

Installation:

  • Main Files - CIBS - Customizable Integrated Backpack System

  • Main Files Canvas Backpacks - FNV

Double-click Canvas Backpacks - FNV - TTW in MO2:

Select the Filetree tab and find the file QwibNewBackpacks.esp then right click and select Hide.

📄 QwibNewBackpacks.esp  ⟵ right click and select Hide

Another Millenia​

This pack now adds over 40+ weapons from Millenias Assets! Now including new custom weapons and unreleased content including 80+ new unique weapons!

Installation:

  • Main Files - Another Millenia

  • Update Files - Another Millenia Update 2.4.6

  • Main Files Hit - Millenia Animations - Part 2

New Vegas Landscape Overhaul Re-Remastered ​

This mod aims to add new life the the Mojave Wasteland by making bland locations more detailed.

Installation:

  • Main Files - New Vegas Landscape Overhaul Re-Remastered

The N.V. Interiors Project​

Brings many new interiors. You need to download the mod below as well.

Installation:

  • Main Files - NVInteriors Core v2.1.1

  • Main Files - NV Interiors Remastered

A World of (Less) Pain - A Lore Friendly AWOP Revision​

Complete stripping down/balancing of A World of Pain

Installation:

  • Main Files - AWOP Revised
